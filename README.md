# Shopper
Android Shopping list app
Written using Android Studio
Utilises SQLite (7 Tables, Shops, Aisles, Products, ProductUsage, Rules, ShopList & Appvalues).


| Feature | Notes |
|:---|:---|
| **Ordered Shopping List**| 3 Tiers of order, **Store**, **Aisle** & **Product** (*within an aisle*)|
| **Rules** | Add Items to the shopping list automatically (can set to prompt before adding ). |
| **Free from Ads and in-App purchases** | Intention is that this will always be the case. |
| **Icon free** | No guessing what a button will do. |
| ***Large Finger Friendly*** | No need to have microscopic sized fingers. |
